## Steps to run
Clone the repo
<code>
git clone https://github.com/jayneeel/Neru-Lending-App.git
</code>

Move inside the project root directory
<code>
cd <i>directory</i>
</code>

Paste the google-servies.json in android/app/

Get the depenedencies
<code>
flutter pub get
</code>

Run in your device
<code>
flutter run
</code>
